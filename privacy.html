ProdHunt - Privacy Policy

Last updated: September 2024

ProdHunt is a browser extension designed to provide users with the top 50 trending products daily from ProductHunt.com. Your privacy is important to us, and this policy explains how ProdHunt handles your data.

Data Collection

ProdHunt does not collect, store, or process any personal data from its users. We do not request or track any personally identifiable information (such as name, email address, or location) when using the extension.

Third-Party Services

ProdHunt retrieves public product data from a third-party service (ProductHunt.com) to display trending products. No personal data is transmitted during this process.

Cookies & Local Storage

ProdHunt does not use cookies. However, the extension may use local storage for caching purposes to enhance performance. This storage does not contain any personal information or user data.

Permissions

ProdHunt requests the following permissions:

  • Storage: To cache product data and improve performance.
  • Active Tab: To allow interaction with the current tab when the user triggers extension actions.

Changes to this Privacy Policy

We may update this policy from time to time. If we make significant changes, we will notify users via the Chrome Web Store page for ProdHunt.

If you have any questions regarding this privacy policy, please feel free to reach out via GitHub issues or the support page on the Chrome Web Store.
